Нужно установить golang-migrate перед тем как перейти к миграциям
Команды для миграций:
migrate create -seq -ext=.sql create_movies_table
migrate create -seq -ext=.sql add_movies_check_contraints
migrate create -seq -ext .sql add_movies_indexes

movielight % migrate -path=./migrations -database=$DB_DSN up

Команда для внесения изменений
migrate -path=./migrations -database="postgres://user:pass@localhost:5432/dbname?sslmode=disable" up

Команда для установки golang-migrate:
go get github.com/golang-migrate/migrate/v4/cmd/migrate@latest

Для version вместо int можно использовать UUID
version = uuid_generate_v4()
